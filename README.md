# DesafioDIO-CopilotEmBranco
Repositório com anotações e exemplos práticos de aprendizagem sobre criação de agentes de IA no Microsoft Copilot Studio, abordando criação de copilot em branco, customização de tópicos, tratamento de erros conversacionais e ajuste da qualidade de respostas utilizando recursos de IA generativa.
